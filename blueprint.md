
# 로또 번호 생성기

## 개요

이 애플리케이션은 사용자가 버튼 클릭으로 매주 로또 번호를 생성할 수 있는 간단한 웹 도구입니다. 생성된 번호는 시각적으로 매력적이고 현대적인 방식으로 표시됩니다.

## 기능

*   **번호 생성:** 사용자는 버튼을 클릭하여 1부터 45까지의 6개 고유한 랜덤 번호를 생성할 수 있습니다.
*   **번호 표시:** 생성된 번호는 정렬되어 화면에 표시됩니다.
*   **반응형 디자인:** 애플리케이션은 모든 화면 크기에서 잘 보이도록 반응형으로 디자인됩니다.

## 디자인

*   **색상 팔레트:** 현대적이고 활기찬 색상 팔레트를 사용하여 사용자 경험을 향상시킵니다.
*   **타이포그래피:** 명확하고 읽기 쉬운 글꼴을 사용하여 생성된 번호와 지침을 표시합니다.
*   **레이아웃:** 중앙에 번호가 표시되고 그 아래에 생성 버튼이 있는 간단하고 직관적인 레이아웃입니다.

## 실행 계획

1.  **`index.html`**:
    *   페이지 제목을 "주간 로또 번호 추첨"으로 변경합니다.
    *   로또 번호와 생성 버튼을 위한 기본 HTML 구조를 만듭니다.
2.  **`style.css`**:
    *   현대적인 디자인 원칙에 따라 애플리케이션의 스타일을 지정합니다.
    *   번호, 버튼 및 전체 레이아웃에 대한 스타일을 포함합니다.
3.  **`main.js`**:
    *   로또 번호를 생성하고 표시하는 JavaScript 로직을 구현합니다.
    *   버튼 클릭 시 6개의 고유한 랜덤 번호를 생성하는 함수를 만듭니다.
